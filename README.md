# 📜 Commands

| Command        | Description           |
| -------------- | --------------------- |
| `/start`       | Start bot             |
| `/help`        | Show commands         |
| `/save`        | Save encrypted memory |
| `/get`         | Search memory         |
| `/all`         | Show all memories     |
| `/delete`      | Delete memory         |
| `/media`       | Show media list       |
| `/viewmedia`   | Retrieve media        |
| `/deletemedia` | Delete media          |
| `/backup`      | Export database       |
